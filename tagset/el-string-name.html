Documentation for NISO-supmat-extension-rec0.dtd

Element: string-name

Name of Person (Unstructured)
Container element for personal names captured with mixed-content where the order of the name components is not enforced.

Attributes

  • @content-type
  • @name-style
  • @specific-use
  • @xml:lang

Content Model

( #PCDATA | degrees | given-names | prefix | surname | suffix )*

May Contain:

  • PCDATA
  • <degrees>
  • <given-names>
  • <prefix>
  • <suffix>
  • <surname>

May be contained in:

  • <contrib>
  • <element-citation>
  • <mixed-citation>
  • <name-alternatives>
  • <person-group>
  • <principal-award-recipient>
  • <principal-investigator>
  • <product>
  • <related-article>
  • <related-object>
