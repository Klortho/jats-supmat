Documentation for NISO-supmat-extension-rec0.dtd

Element: sme:resource

Description of and pointer to an external file or folder (directory on a filesystem) that holds one or more objects (<sme:resource> or <sme:item>). * In the JATS SME extension, <graphic> and <media> should be replaced by <sme:resource>.
