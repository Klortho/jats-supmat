Documentation for NISO-supmat-extension-rec0.dtd

Element: table

Table (NISO JATS table model)
The rows and columns part of a full table. (This element is based on and intended to be converted easily to the XHMTL table element.)

Attributes

  • @border
  • @cellpadding
  • @cellspacing
  • @content-type
  • @frame
  • @id
  • @rules
  • @style
  • @summary
  • @width

Content Model

( ( col* | colgroup* ), ( ( thead?, tfoot?, tbody+ ) | ( tr+ ) ) )

May Contain:

  • <col>
  • <colgroup>
  • <tbody>
  • <tfoot>
  • <thead>
  • <tr>

May be contained in:

  • <alternatives>
  • <table-wrap>
