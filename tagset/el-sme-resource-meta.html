Element: sme:resource-meta

Container element for the metadata concerning a file or folder (directory on a filesystem).

Content Model

( sme:filename?, sme:format?, sme:format-registry?, sme:validity?, size?, sme:fixity?, sme:creating-application*, sme:rendering-application* )

May Contain:

  • <size>
  • <sme:creating-application>
  • <sme:filename>
  • <sme:fixity>
  • <sme:format>
  • <sme:format-registry>
  • <sme:rendering-application>
  • <sme:validity>

May be contained in:

  • <sme:resource>
