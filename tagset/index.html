Documentation: NISO-supmat-extension-rec0.dtd

This is an extension of NISO JATS 1.0 (Journal Publishing Tag Set) to accommodate the NISO Recommended Practice for Supplementary Materials. Because it is an extension, any Journal Publishing 1.0 article will be valid here.

Some interesting starting points are:

  • <article>
  • <fig>
  • <supplementary-material>
  • <ext:resource>
  • <ext:item>
