Documentation for NISO-supmat-extension-rec0.dtd

Element: sme:item

Container element for the metadata concerning one object such as a single figure, a single table, or a single graphic.

Attributes

  • @id
  • @rid

Content Model

( ( ( object-id )*, sme:version?, label?, ( caption )*, ( alt-text | long-desc | email | ext-link | uri )*, ( ( disp-formula | disp-formula-group | chem-struct-wrap | kwd-group | disp-quote | speech | statement | verse-group | contrib-group | sme:summary | table-wrap | p | sme:provenance | sme:preservation | publisher | def-list | list ) )*, ( attrib | permissions )* ) )
